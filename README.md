# yewu-skill (ui组业务组件)

Codex skill：按 ui组业务组件 规范生成中后台界面（工作台、列表、详情等）。

## 内容

- `arco-admin-ui/` — skill 本体（安装时复制此目录）
- `INSTALL.md` — 安装说明

## 展示名

- display_name: **ui组业务组件**
- name: `arco-admin-ui`（调用：`$arco-admin-ui`）

## 快速安装

将 `arco-admin-ui` 复制到：

```text
%USERPROFILE%\.codex\skills\arco-admin-ui
```

新开 Codex 对话后使用。

## 规范要点

- 顶栏：通栏深蓝 `#004EA2`（系统名按业务填写）
- 侧栏：白底，选中浅蓝 `#E6F1FF`
- 主按钮：`#165DFF`
- 参考页：工作台、表格列表、详情（有/无步骤条）见 `arco-admin-ui/references/examples/`
